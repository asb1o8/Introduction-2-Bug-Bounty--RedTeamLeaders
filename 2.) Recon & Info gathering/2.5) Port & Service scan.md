Enable -sC to run default scripts (great for detecting vulnerable services).


#### ðŸŒ³ Understand Common Non-HTTP Ports:-
Notable ports and associated services:
- 21: FTP
- 22: SSH
- 23: Telnet
- 25, 465, 587: SMTP
- 110, 143: POP/IMAP
- 3306: MySQL
- 5432: PostgreSQL
- 6379: Redis
- 9200: Elasticsearch
- 27017: MongoDB  
- 8080, 8443, 8000, 8888: Dev servers

Knowing these helps prioritize targets.


#### ðŸŒ³ Banner Grabbing and Service Identification

Identify what is running using:

    nmap -sV
    nc (Netcat)
    curl or wget on web-based ports

Banner info reveals:

    Software versions
    Server types
    Leaky headers (e.g., X-Powered-By, Server)
