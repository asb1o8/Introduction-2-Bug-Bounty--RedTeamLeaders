### ✨ Platforms used for Bug Bounty

#### Platforms:-
- Hackerone
- Bugcrowd
- Intigriti
- YesWeHack

#### Choosing the Right Platform as a Beginner:-

- If you want access to big names and learn fast under pressure: Start with HackerOne.
- If you value gradual access to private programs and structured learning: Go with Bugcrowd.
- If you’re based in Europe or want personalized triage feedback: Try Intigriti.
- If you’re interested in public sector, multilingual or EU-centric programs: Explore YesWeHack.
